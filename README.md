# 💼 Portfolio Personale

Benvenuto nel repository del mio **Portfolio Personale**.

Questo progetto raccoglie i miei lavori, le competenze tecniche e il mio percorso come sviluppatore web. L'obiettivo è creare una vetrina moderna, responsive e intuitiva che permetta di conoscere rapidamente chi sono, le tecnologie che utilizzo e i progetti che ho realizzato.

---

## ✨ Caratteristiche

* 📱 Design completamente responsive
* 📲 Approccio **Mobile-First**
* 🎨 Interfaccia moderna e pulita
* ⚡ Navigazione fluida
* 📂 Sezione dedicata ai progetti
* 👨‍💻 Presentazione personale
* 📞 Contatti e collegamenti ai social
* 🚀 Ottimizzato per desktop, tablet e smartphone

---

## 🛠️ Tecnologie utilizzate

* HTML5
* Tailwind CSS
* JavaScript

---

## 🚀 Avvio del progetto

Clona il repository:

```bash
git clone https://github.com/AntonyDev03/portfolio.git
```

Accedi alla cartella del progetto:

```bash
cd portfolio
```

Se utilizzi Tailwind CSS tramite CLI:

```bash
npm install
npm run dev
```

In alternativa, apri semplicemente il file `index.html` nel browser.
